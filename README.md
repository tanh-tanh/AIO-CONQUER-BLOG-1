# 4.Quy trình 5 bước của một dự án Machine Learning
## Bước 1: Định nghĩa bài toán (Problem Definition):
Mục tiêu là gì? (Dự đoán doanh số, phân loại ảnh?).
Loại bài toán nào? (Supervised hay Unsupervised?).
## Bước 2: Thu thập và Xử lý dữ liệu (Data Collection & Preprocessing):
Tìm dữ liệu ở đâu? (Kaggle, Database, API).
Làm sạch dữ liệu (Xử lý dữ liệu thiếu - Missing values, dữ liệu nhiễu - Noise).
Chuẩn hóa dữ liệu (Feature Scaling).
## Bước 3: Lựa chọn và Huấn luyện mô hình (Model Selection & Training):
Chia tập dữ liệu: Train set (để học) và Test set (để thi).
Chọn thuật toán phù hợp và cho máy "học".
## Bước 4: Đánh giá mô hình (Model Evaluation):
Mô hình dự đoán có đúng không?
Các chỉ số đánh giá cơ bản: Accuracy (Độ chính xác), Precision, Recall, MSE (Mean Squared Error).
## Bước 5: Tinh chỉnh và Triển khai (Improvement & Deployment):
Tinh chỉnh tham số (Hyperparameter tuning) để kết quả tốt hơn.
